# Student Manager — React CRUD | Soham

A simple CRUD app to manage student records.

## Features
- **Create** — Add new student with form validation
- **Read** — View all students in a table, search by name/email, filter by branch
- **Update** — Edit any student via a popup modal
- **Delete** — Remove a student with confirm dialog

## Run
```bash
npm install
npm run dev
```
